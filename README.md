Random Password Generator

A simple Random Password Generator built using HTML, CSS, and JavaScript. This project helps users create strong and secure passwords instantly.

Features

Generates random passwords

Uses letters, numbers, and special characters

Simple and user-friendly interface

Fast and lightweight

 Technologies Used

HTML – Structure of the web page

CSS – Styling and layout

JavaScript – Password generation logic

 Project Structure
├── index.html
├── style.css
└── script.js

How to Use

Clone or download the repository

Open index.html in your browser

Click the generate button to create a password

Purpose

This project is created to practice basic front-end web development skills and understand how JavaScript can be used to generate dynamic content.
